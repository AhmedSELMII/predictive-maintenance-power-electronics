### Run the Complete Pipeline

```bash
# 3. Train models
python src/train_models.py
```

### Use Trained Models for Prediction
```python
import joblib
import pandas as pd

# Load trained model
classifier = joblib.load('models/failure_classifier.pkl')
regressor = joblib.load('models/rul_regressor.pkl')

# Load new sensor data
new_data = pd.read_csv('your_sensor_data.csv')

# Predict failure
will_fail = classifier.predict(new_data)
hours_remaining = regressor.predict(new_data)

print(f"Will fail soon: {will_fail[0]}")
print(f"Hours until failure: {hours_remaining[0]:.1f}")
```
